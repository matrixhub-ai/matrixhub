# 模型设置

仅具有管理权限的成员可编辑 **模型设置** 标签页。

![模型设置](./images/model-setting.jpg)

## 设为热门模型

1. 进入目标模型详情页，切换到 **模型设置** 。

1. 勾选 **设为热门模型** 。

1. 保存设置后，在 **模型仓库** -> **推荐模型** 中确认模型已展示。

    ![热门模型](./images/hot-model.jpg)

:::note

- 设为热门后，平台用户可在推荐列表中看到该模型。

:::

## 删除模型

1. 进入目标模型详情页，切换到 **模型设置** 。

1. 点击 **删除模型** 。

1. 在确认弹窗中输入项目名称并提交删除。

:::warning

删除模型后不可恢复，请在操作前确认已完成备份。

:::
